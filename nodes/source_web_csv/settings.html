Separator Define how columns are separated in CSV. If tabulator is used, then check "Tab separated" checkbox. Character encoding If you get weird characters, then try different encoding. Field names The first row of CSV file usually holds the names of the columns. If not, then uncheck this. Trim values Selecting this removes any extra spaces from values. Empty lines If this is checked, empty lines are not imported. Multiple value separator If CSV includes multiple values, then those values can be splitted by defining the separator here. Authentication If CSV is protected with basic authentication, then you can set credentials here.
