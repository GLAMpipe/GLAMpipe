Backend: Sent email to any email address

This node can be used for confirmation emails in Shibboleth authenticated services.Message content, subject and receiver can not be altered via REST.
In order to use this node, one must have an ability to send unauthorised emails. Shibboleth header name is defined in config.js by "shibbolethHeaderId".
You can use this in subject and message texts by using double square brackets ("[[variable]]")
