Backend: Sent email to any email address

This node can be used for sending mail. Message content, subject and receiver are defined in settings and so they can be set via REST.
In order to use this node, one must have an ability to send unauthorised emails.
You can use dynamic document values in text by writing field name surrounded by square brackets like this: [[dc_title]]
