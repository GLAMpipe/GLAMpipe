Backend: Sent email to any email address This node can be used for sending mail. Message content, subject and receiver are defined in settings and so they can be set via REST.
In order to use this node, one must have an ability to send unauthorised emails. Message settings You can use dynamic document values in subject and message fields by writing field name surrounded by square brackets like this: [[dc_title]] Document fields You can use "[[data_fields]]" variable in message in order to render document fields to a message body. Here you can define what those fields are.
