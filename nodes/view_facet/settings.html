View title Set the title of the facets page. Facets Choose what fields you want see as facets, which side of the page you want see them and whether they are collapsed or not. Item rendering Here you can define what record fields are rendered as a result.
If you check "link", then the field value is rendered as a link. If value is not a full link, you can write start of the link in the last input.

When you are ready, run node (click "Export data") and then open your faceted view
