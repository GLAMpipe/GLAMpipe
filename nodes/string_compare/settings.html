Compare Copy this if there IS match Here you can set static string OR dynamic field that should be written to output field when there is a match. Copy this if there IS NO match Here you can set static string OR dynamic field that should be written to output field when there is no match. Match mode Normal match means that if there is a match, copy selected value to output field.
Emtpy match means that if one of the match fields is empty, then it is considered as a match and select value is copied to output field. This useful when you want to make sure that there is always a value in copy field even if another field was empty. Trim Values get trimmed before comparision if this is checked.
