Match mode Normal match means that if there is a match, copy selected value to output field.
Copy non-empty means that if one of the match fields is empty, then the non-empty values is copied to output. This useful when you want to make sure that there is always a value in ouput. Compare Choose a comparasion field. Copy this if there IS match Here you can set static string OR dynamic field that should be written to output field when there is a match. Copy this if there IS NO match Here you can set static string OR dynamic field that should be written to output field when there is no match. Trim Values get trimmed before comparision if this is checked.
