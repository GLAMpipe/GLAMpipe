Backend: Sent email to Shibboleth user

This node allows you to send emails via REST api to Shibboleth user. In ordet to use this node, one must have an ability to send unauthorised emails. Shibboleth header name is defined in config.js by "shibbolethHeaderId".
This can be used for confirmation emails in Shibboleth authenticated services.
You can use this in subject and message texts by using double square brackets ("[[variable]]")
