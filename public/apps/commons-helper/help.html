Commons Helper: Help page

What is Commons Helper?

Commons Helper is a tool that simplifies the Commons uploads with GLAMpipe. Helper creates the node structure needed for Commons uploads project for you.

Prepare your metadata

GLAMpipe Commons helper requires CSV file. You can export CSV file from Excel and Libreoffice.

  1. Make sure that the first row in your spreadsheet contains field labels.
  2. Make sure that labels match Commons template fields. (see templates)
  3. Metadata file must include direct links to the files.
