Welcome to GLAMpipe!

GLAMpipe is a tool for cultural data processing.

What is GLAMpipe

  • operates on collections, documents and nodes (not tables and cells)
  • for tasks that require human intervention (i.e. task that can not be fully automatised)
  • is able to handle multiple values in different languages

What's so special in cultural data?

In technicals sense the main difference when compared to scintific data, is that cultural data often has multiple values per field and that there are language codes.
